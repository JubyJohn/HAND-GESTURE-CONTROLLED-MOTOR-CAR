# HAND-GESTURE-CONTROLLED-MOTOR-CAR

## AIM
TO CONTROL RC MOTOR CAR WITH HAND MOVEMENTS - ESP32 NOW APPLICATION


## COMPONENTS

- ESP32 x2
- DC motor x4
- L293D motor driver
- 12V adapter
- Accelerometer
- Jumper Wire
- USB cable
